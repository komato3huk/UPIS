

## Requirements

Ansible 2.7.0+

Python 3.7.0+

## Usage

Clone repo
```
git clone https://github.com/xvost/Ansible_proxies.git
```
Change dir
```
cd ./Ansible_proxies
```
Rename data.conf
```
cp ./data.conf.example ./data.conf
```
Edit data.conf
```
editor ./data.conf
```
Start creation script 
```
./start_install.py  
```
